# Portmation for ATOLL-AB devices.

# About the SoC
- Manufactured by Qualcomm Inc.

- 2x Kryo 465 Gold (2.3GHz)

- 6x Kryo 465 Silver (1.8GHz)

- Adreno 618 GPU

- Snapdragon 720G, SM6150-AB or SM7125-AB

- 8 nanometer, up to 2.4GHz

# About the device
- Xiaomi, Inc. named the board as ATOLL-AB, community ported multiple ATOLL ROMs to miatoll devices.

- miatoll is a unified codename for Redmi Note 9 Pro/Pro Max/9S/Note 10 Lite/Poco M2 Pro.

- Codenames are: joyeuse, excalibur, curtana, curtana_in and gram (afaik).

- Device has MIUI 11 (Android 10) in factory, but Xiaomi gave MIUI 14 update which is Android 12, so 2 android upgrades.
